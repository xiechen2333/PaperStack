# 📚 PaperStack

**PaperStack** 是一款由 **Vibe Coding** 构建的本地优先（Local-first）科研文献管理工具。
它无需联网，无需配置后端，所有数据安全存储在您的浏览器中，旨在为您提供纯粹、流畅且隐私安全的阅读与笔记体验。

## ✨ 主要功能

- **🔒 数据完全本地化**：基于 IndexedDB，数据只存本地。支持一键 JSON 导入/导出，数据主权归你。
- **📝 沉浸式笔记**：内置 Markdown 编辑器，完美支持 **LaTeX 公式** ($E=mc^2$)，流畅记录科研灵感。
- **🗂 灵活管理**：支持无限层级文件夹、拖拽排序与多维分类，轻松构建专属知识树。
- **⚡️ 极速性能**：毫秒级全文检索，配合虚拟列表技术，管理海量文献依然丝滑。

##  快速开始

**1. 安装依赖**
   ```bash
   npm install
   ```

**2. 启动应用**
   ```bash
   npm run dev
   ```
   打开浏览器访问 `http://localhost:5173` 即可开始使用。

**3. 构建部署** (可选)
   ```bash
   npm run build
   ```

# PaperStack 📚

> **👋 大家好！本项目是一个由 AI 完全辅助生成的开源小工具。**

**PaperStack 并不是用来替代 Zotero 等文献管理软件的。** 它的定位是文献阅读器的“最佳辅助”——帮你把阅读时的灵感、评分和核心思绪，变成清晰的结构化记录。

平时用 Zotero 看论文，或许会高亮和批注往往到处都是，过一阵子就想不起来重点。这个小工具就是为了解决这个问题：**方便你随时记录下各种 IDEAS、核心方法或是对一篇论文的打分与评价。** 

💡 **最省力的终极用法**：读完论文后，你可以直接让 AI（比如 ChatGPT/Claude）帮你生成一份 Markdown 格式的核心总结，然后一键粘贴到这里，再配上你自己的评分和两句想法，就完成了一篇高质量的阅读记录！

## ✨ 核心亮点

- **专注笔记与灵感提取**：内建四宫格结构，也支持纯净的 Markdown 粘贴。无论是自己写还是用 AI 帮写，都能完美兼容。
- **多维度检索与状态追踪**：
  - **阅读状态与评分**：支持标记“待读”、“在读”、“已读”，并能为其打分（1-10分）。
  - **时间追踪不遗忘**：系统会精准记录你具体哪一天打的分（即读完的时间），结合“最近阅读”功能，你永远清楚最近自己看了些什么。
  - **自由标签**：通过自建标签库（Tags）把跨领域的论文串联起来。
- **保护隐私，数据纯本地**：完全在你的浏览器里运行（基于 IndexedDB），无需注册，数据不用上传云端。
- **随时备份，带走数据**：可以一键导出所有的笔记数据（JSON 格式），方便更换设备或做长期存档。
- **护眼的深色模式**：全局适配了高对比度的护眼深色模式，深夜读 paper 不刺眼。

## 🚀 怎么运行？

这是一个前端单页应用（React + Vite），需要你的电脑上有 Node.js 环境。

1. **装依赖**：
   ```bash
   npm install
   ```
2. **跑起来**：
   ```bash
   npm run dev
   ```

*(💡 如果你是 Windows 并且配置好了环境，直接点开目录下的 `启动文献记录.bat` 也能一键运行！)*

## 💡 怎么用最顺手？

1. **建大纲**：点击左下角按钮进入“管理模式”，建立你的大研究方向（比如：多模态、时间序列）。
2. **读完即记**：配合 Zotero 读完 PDF，切过来点“新文献”，如果是泛读，直接贴上 AI 生成的 Markdown 总结；如果是精读，自己写一写灵感和方法。
3. **记得备份**：每天下班前，随手点一下右上角的“备份”按钮，妥善保存你的知识结晶。
